(주)알포터 코딩가이드라인

 

본 페이지는 (주)알포터에서 코딩을 함에 있어 준수해야할 내용을 정립, 기록한 것입니다.

당사의 모든 개발자들은 본 내용을 참고하여 개발을 진행해 주시기 바랍니다.

 

C# 코딩 가이드라인

기본적으로 C#(Java, TypeScript 포함)은 파스칼 표기법을 사용합니다.

  1. 파라미터의 인자는 소문자로 시작합니다.

        
// C# 코드 예시
public class ExampleClass
{
  public int Add(int a, int b)
  {
      return a + b;
  }
}

JavaScript 코딩 가이드라인

기본적으로 javascript는 카멜 표기법을 사용합니다.

  1. 파라미터의 인자는 소문자로 시작합니다.

        
// JavaScript 코드 예시
function add(a, b) {
  return a + b;
}

공통사항

다음 내용은 언어에 상관없이 공통적으로 적용됩니다.

  1. 기본적으로 모든 명명요소는 두가지 이상의 단어를 조합하여 사용합니다.
  2. 단어는 축약하여 사용하지 않고, 되도록 해당 단어의 전체 글자를 모두 표기합니다.
  3. 두번째 단어부터는 첫글자를 대문자로 하는 것을 원칙으로 합니다.
  4. 단일 단어를 사용하는 경우에는 해당 단어의 사용이 합당한지 충분히 고려되어야 합니다.
  5. 축약하여 사용하여도 축약된 단어가 어떤 단어인지 누구든지 추측이 가능한 경우에는 축약된 단어를 사용할 수 있습니다.
